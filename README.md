# UPI Fraud Detection System

This project detects suspicious UPI transactions using Machine Learning and displays alerts through a React dashboard.

## Features
- Detect fraudulent transactions
- Transaction monitoring dashboard
- Risk score calculation
- Alert system for suspicious payments

## Tech Stack
Frontend: React.js  
Backend: Python / Flask (or Node.js if used)  
Machine Learning: Scikit-learn / TensorFlow  
Database: MongoDB / MySQL

## Installation

1. Clone the repository
git clone https://github.com/Bavadharani-M/upi-fraud-detection.git

2. Install dependencies
npm install

3. Run the project
npm start
